## Features

- Drag and drop file upload
- Support for multiple file types:
  - Documents (PDF, DOCX, TXT, etc.)
  - Images (JPG, PNG, WEBP, etc.)
  - Videos (MP4, AVI, MKV, etc.)
  - Audio (MP3, WAV, FLAC, etc.)
  - Archives (ZIP, RAR, 7Z, etc.)
- Smart format recommendations
- Quality settings for image/video conversions
- Modern UI with glassmorphism design

## Requirements

- .NET 8.0 or higher
- Windows OS

## Usage

1. Launch the application
2. Upload a file by clicking the upload button or drag & drop
3. Select the desired output format
4. Adjust quality settings if needed
5. Click on the format to start conversion
6. Download your converted file

## Note

- **Video to GIF conversion**: Requires FFmpeg to be installed on your system. Download from [ffmpeg.org](https://ffmpeg.org/download.html)
- Other conversions are currently simulated (UI prototype only)
